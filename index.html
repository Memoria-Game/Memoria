About

Memoria is a combinaison of memory and path-finder game built by Heig-vd students during a Scala project. The front end is developped with bootstrap and phaser.js whereas the back-end is developped with Scala play framework. The principe of the game is simple, you need to reach the green square without touching the walls(black squares). The more you succeed, the more difficult it is. Collect some bonus to facilitate your travel, you need 5 of the same type to use it
